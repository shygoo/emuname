# emuname

#### Java NES emulator with a plugin API for hacking

Not many games work yet.

![Image](https://i.gyazo.com/fac95a4625da8c094a8bbacd3f838a12.gif)

#### Default button mappings
```
A ....... m
B ....... n
SELECT .. q
START ... e
UP ...... w
DOWN .... s
LEFT .... a
RIGHT ... d
```
